طراحی سایت با HTML و CSS

وب‌سایت‌ها بخشی جدایی‌ناپذیر از زندگی روزمره ما هستند. از خرید آنلاین تا شبکه‌های اجتماعی، همه‌چیز روی وب اتفاق می‌افتد. اما آیا تا به حال فکر کرده‌اید که این وب‌سایت‌ها چطور ساخته می‌شوند؟ در این بخش، با مفاهیم پایه‌ای طراحی سایت آشنا می‌شویم و یاد می‌گیریم که چطور با HTML و CSS یک صفحه وب ساده ایجاد کنیم.

دانش پایه طراحی وب

طراحی وب فقط به HTML و CSS محدود نمی‌شود. برای ساخت وب‌سایت‌های تعاملی و پویا، باید با مفاهیم پیشرفته‌تری آشنا شوید. در اینجا برخی از این مفاهیم را بررسی می‌کنیم:

  • DOM (Document Object Model): DOM یک رابط برنامه‌نویسی برای HTML و XML است. با استفاده از DOM، می‌توانید ساختار یک صفحه وب را تغییر دهید یا به آن دسترسی داشته باشید.
  • BOM (Browser Object Model): BOM به شما امکان دسترسی به ویژگی‌های مرورگر، مثل پنجره‌ها و تاریخچه، را می‌دهد.
  • جاوااسکریپت: جاوااسکریپت یک زبان برنامه‌نویسی است که به صفحات وب تعامل می‌دهد. با استفاده از جاوااسکریپت، می‌توانید عناصر صفحه را تغییر دهید، رویدادها را مدیریت کنید و با سرور ارتباط برقرار کنید.
  • فریم‌ورک‌ها (مانند React): فریم‌ورک‌هایی مثل React به شما کمک می‌کنند تا وب‌سایت‌های پیچیده و پویا را راحت‌تر بسازید.
  • رندر صفحه: مرورگرها کدهای HTML، CSS و جاوااسکریپت را دریافت کرده و آن‌ها را به یک صفحه وب تبدیل می‌کنند. این فرآیند را رندر کردن می‌گویند.

HTML و CSS پایه

HTML و CSS پایه‌ترین ابزارهای طراحی وب هستند. در اینجا برخی از تگ‌ها و ویژگی‌های اصلی آن‌ها را بررسی می‌کنیم:

  • تگ‌های اصلی HTML:
    • <a>: برای ایجاد لینک.
    • <div>: برای گروه‌بندی عناصر.
    • <table>: برای ایجاد جدول.
    • <span>: برای گروه‌بندی متن‌ها.
    • <tr> و <td>: برای ایجاد سطر و ستون در جدول.
    • <ul> و <li>: برای ایجاد لیست‌های نامرتب.
  • ویژگی‌های اصلی CSS:
    • color: برای تغییر رنگ متن.
    • font-family: برای تغییر فونت.
    • margin و padding: برای تنظیم فاصله‌ها.
    • display: برای کنترل چیدمان عناصر.

نمونه کد HTML

در زیر یک نمونه کد HTML ساده آورده شده است: 


<!DOCTYPE html>
<html lang="fa">
<head>
    <meta charset="UTF-8">
    <title>صفحه من</title>
</head>
<body>
    <h1>به وب‌سایت من خوش آمدید!</h1>
    <p>این یک پاراگراف ساده است.</p>
    <a href="https://example.com">این یک لینک است</a>
    <img src="image.jpg" alt="عکس من">
</body>
</html>

نمونه کد CSS

در زیر یک نمونه کد CSS ساده آورده شده است: 


body {
    font-family: 'Roboto', sans-serif;
    background-color: #f8f9fa;
    color: #333;
}

h1 {
    color: #1abc9c;
    text-align: center;
}

p {
    font-size: 1.2rem;
    line-height: 1.6;
}

نتیجه ترکیب HTML و CSS

وقتی HTML و CSS با هم ترکیب می‌شوند، یک صفحه وب زیبا و کاربردی ایجاد می‌شود. در تصویر زیر، نتیجه ترکیب کدهای بالا را مشاهده می‌کنید.
